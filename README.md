Esto más que un proyecto serio es un primer contacto con las APIs y Javascript.

Se selecciona el producto de la máquina y se le da al botón de aceptar para quitar 1u. de la base de datos.

Puedes cambiar al modo reponer, el cual primero pedirá el número del artículo y después deberás introducir la cantidad deseada. Se modificará el stock en la base de datos.
